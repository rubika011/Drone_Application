#Drone Application

####Technologies Used
- Java
- Spring Boot
- Maven
- Rest API
- JPA
- H2(In memory Database)

####Prerequisite:
- Java 19
- Intellij or Eclipse IDE
- PostMan (for testing API)

####Build
- Clone the git repo https://github.com/rubika011/Drone_Application.git
- Import the project in IDE and Run the spring boot application
